# Summary
This is a Trial Task for *Kameleoon*.

A simple REST API written on Java, using Spring Boot and H2.

My API allowing:

- POST /users
- POST /quotes
- GET /quotes/{id}
- GET /quotes/random
- GET /quotes/top10
- GET /quotes/bottom10
- PUT /quotes/{id}
- DELETE /quotes/{id}
- POST /votes
- GET /votes/{quoteId}
